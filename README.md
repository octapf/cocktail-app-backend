# Cocktail App Backend

## Description

The server to a Wiki App to learn about the most famous cocktails in the world

## Build with

- Typescript
- Node
- Express
- Mongo
- Vercel

## Getting Started

### Installing & Excecuting

1. Clone the repo
   ```sh
   git clone https://github.com/octapf/cocktail-app-backend.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

3. Run the project

   ```sh
   npm run dev
   ```
